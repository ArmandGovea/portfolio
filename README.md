#  Portfolio Website

A modern, interactive portfolio showcasing web development skills with a terminal-style hero section, smooth animations, and Easter eggs.

**Live Demo:** [armandgovea.vercel.app]([https://portfolio.armandgovea.com](https://armandgovea.vercel.app))

---

##  Features

 **Terminal-Style Hero** — Interactive typing animation  
 **Dark Theme** — Gold & cyan modern aesthetic  
 **Smooth Animations** — Scroll transitions and fade-ins  
 **Interactive Elements** — Hover effects and smooth scrolling  
 **Easter Eggs** — Konami code activation, Matrix rain effect  
 **Project Showcase** — Links to live projects and GitHub repos  
 **Contact Section** — Email and social links  
 **Fully Responsive** — Mobile-first design  

---

##  Tech Stack

- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Animations:** CSS transitions, JavaScript DOM manipulation
- **Hosting:** Vercel
- **Version Control:** Git & GitHub

##  Quick Start

### Local Development

1. **Clone the repository:**
```bash
git clone https://github.com/ArmandGovea/portfolio.git
cd portfolio
```

2. **Run locally:**
```bash
# Option A: Python
python -m http.server 8000

# Option B: VS Code Live Server
# Right-click index.html → Open with Live Server

# Open http://localhost:8000
```

##  Deployment

### Deploy to Vercel

1. Push to GitHub:
```bash
git add .
git commit -m "Portfolio website"
git push origin main
```

2. Go to [vercel.com](https://vercel.com)
3. Click "New Project" → Import from GitHub
4. Select this repository
5. Click Deploy!

Your portfolio is now live! 🎉
